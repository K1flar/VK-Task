# VK-Task

Rest API film library.

## Getting Started

1. Build the Docker containers:
```bash
make build
```

2. Start the application:
```bash
make start
```

3. Apply database migrations:
```bash
make migrate-up
```

## Test data

Load test data into the database:
```bash
make testdata
```
